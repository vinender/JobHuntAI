# JobHuntAI

AI-powered job search pipeline.

## Overview
JobHuntAI automates and optimizes the job search process using powerful AI models. 

## Setup
1. Ensure you have Node.js installed.
2. Run `npm install` to install dependencies.
3. Configure your settings in the `config/` directory.

## Usage
- Run `npm run verify` to verify the pipeline.
- Other scripts can be found in the `package.json` file.
